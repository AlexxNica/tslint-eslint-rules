## no-inner-declarations (eslint: [no-inner-declarations](http://eslint.org/docs/rules/no-inner-declarations))

disallow function or variable declarations in nested blocks (recommended)

### Usage

```json
"no-inner-declarations": [
  true,
  "functions"
]
```

```json
"no-inner-declarations": [
  true,
  "both"
]
```
