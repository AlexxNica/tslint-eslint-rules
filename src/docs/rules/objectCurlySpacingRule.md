## object-curly-spacing (ESLint: [object-curly-spacing](http://eslint.org/docs/rules/object-curly-spacing))

require or disallow padding inside curly braces

### Usage

```json
"object-curly-spacing": [
    true,
    "always"
  ]
```

```json
"object-curly-spacing": [
    true,
    "never"
  ]
```
