## no-extra-boolean-cast (ESLint: [no-extra-boolean-cast](http://eslint.org/docs/rules/no-extra-boolean-cast))

disallow double-negation boolean casts in a boolean context (recommended)

### Usage

```json
"no-extra-boolean-cast": true
```
