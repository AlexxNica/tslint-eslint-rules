## block-spacing (ESLint: [block-spacing](http://eslint.org/docs/rules/block-spacing))

disallow or enforce spaces inside of single line blocks

### Usage

```json
"block-spacing": [
    true,
    "always"
  ]
```

```json
"block-spacing": [
    true,
    "never"
  ]
```
